<h1 align="center">MatterVibe</h1>
<p align="center">为中国执业律师打造的本地案件管理工作台<br>Local-first matter management for litigators · macOS & Windows</p>

---

MatterVibe 是一款为中国执业律师量身打造的、以案件为核心的本地桌面工作台，由律师本人以 AI Vibe Coding（AI 氛围编程）模式与 Anthropic Claude 协作独立开发。

它摒弃传统法律软件的冗余设计：首创永远全展开的「案卷封皮」视图，案号、承办法官、期限红线一眼尽收；五套深度定制的办案模板（执行、民商事、刑事黄金 37 天、非诉、行政诉讼）开箱即用，内置博弈节点与程序红线；办案日志与文书邮寄记录随手沉淀为工作台账；按案件开关的期限提醒自动扫描封皮日期，并内置法定节假日与调休的届满日顺延规则。

随附完全离线的律师工具箱：诉讼费、保全费、风险代理费、仲裁费（费率表可校准）、法定期限届满日、金额大写等高频刚需，输入即出结果。

依托本地优先的 SQLite 单文件架构，全部案卷与当事人信息只存于你自己的设备，不向任何服务器上传；可选的本地 AI 接口（默认关闭、令牌鉴权、支持只读）让 Marvis、小龙虾等 AI 助手安全地为你拆解案情、自动建档。

本软件已开源，按 Apache License 2.0 向法律同行免费提供——做一件如纸笔般纯粹、行云流水的数字化办案工具。欢迎使用、提出建议、贡献模板与代码。

## 主要功能

- **案件看板**：永远全展开的案卷封皮 ＋ 横向办案阶段列，任务划销带纸笔质感音效，拖拽流转
- **五套出厂模板**：执行（立体化财产外调）、民商事、刑事（黄金 37 天）、非诉、行政（期限阻击／举证倒置监控），模板可编辑、可自建、自动备份可还原
- **期限提醒**：按案件开关，自动扫描封皮日期；内置 2025–2026 法定节假日与调休，届满日依法顺延；顶栏铃铛汇总＋系统通知
- **办案日志／邮寄记录**：时间线＋工时统计；快递三选＋材料智能标签，一键复制整案台账
- **律师工具箱**（独立窗口，全离线）：诉讼费／保全费、风险代理费、仲裁费（贸仲·北仲，费率可校准持久化）、法定期限届满日（14 种预设）、人民币大写、文本去重、字数统计
- **案件文件夹直达**：一键在访达打开案卷文件夹，新建案件可自动建夹
- **本地 AI 接口**：默认关闭；开启后 Marvis／小龙虾（OpenClaw）等 AI 助手可经 127.0.0.1＋令牌读写案件（支持只读模式），详见《AI接口说明.md》
- **数据主权**：SQLite 单文件，全部数据仅存本机；一键备份整库

## 运行与打包

```bash
npm install
npm start          # 开发运行
npm run pack       # 打包 macOS .app（dist/）
npm run pack:win   # 打包 Windows x64（dist/）
```

详细图文步骤见《零基础三步运行指南.md》。
