# MatterVibe 软件说明书

> MatterVibe 是一款为中国执业律师打造的本地、开源案件管理软件。本说明书介绍它的
> 全部功能与使用方法。软件内「设置 → 使用帮助」有精简版速查。

---

## 一、它是什么、不是什么

**是什么**：一个把每个案件做成"看板"的桌面工具。每个案件 = 一列案卷封皮（案号、当事人、
法院、承办法官、关键期限等要素）+ 若干办案阶段（如收案准备、立案管辖、证据庭前、开庭、
裁判后续）+ 阶段下可勾销的任务卡片。你像在真实卷宗上推进工作一样，逐条划销。

**不是什么**：不是云端 SaaS，不上传任何数据。全部案卷与当事人信息只存在你这台电脑上
（一个 SQLite 单文件数据库）。没有账号体系、没有订阅、没有广告。

**核心理念**：如纸笔般纯粹。律师的核心资产是判断力和案卷的有序，软件只做"让卷宗一目了然、
期限不遗漏、思路可沉淀"这几件事，不做花哨而无用的功能。

## 二、快速上手（5 分钟）

1. **新建案件**：点左下角「＋ 新建案件」，选一套模板（民商事 / 刑事 / 行政 / 执行 / 非诉），
   立刻生成一个带阶段和任务的案件。
2. **填封皮**：在案件信息列里填案号、当事人、法院、关键期限等。点击别处自动保存，无需手动保存。
   底部有「复制案件信息」可一键复制为规整文本，方便发给当事人或同事。
3. **推进任务**：在阶段列里勾销已完成的任务、点「＋ 添加任务」补充新的；可拖拽调整、删除。
4. **开启提醒**：点信息列的铃铛图标开启该案提醒。软件会自动识别封皮里的日期（如"开庭日期：
   2026-07-01"），临近时在顶栏铃铛和系统通知里提醒，并汇入首页与日历。

## 三、功能详解

### 案件看板
- 案卷封皮永远全展开，要素一目了然；要素可增删改、可调整。
- 办案阶段横向排列，每阶段下是任务卡片，勾销带纸笔质感。
- 案件可设置图标、关联本地文件夹（一键打开案件资料文件夹）、克隆、归档、删除。

### 办案模板
- 内置五套：民商事、刑事辩护、行政、执行、非诉，体现常见办案程序。
- 民商事等为"不分立场的通用骨架"，代理原告或被告都适用。
- 可在「设置 → 模板编辑」自定义；可导出/导入模板（.json），与同行分享办案思路。

### 首页 · 待办总览
跨案件汇总：今日开庭、本周开庭、近 7 日临期、已逾期未办，以及待办清单与今日安排。

### 日历
月视图按日显示开庭、举证、调解、自定义事件，标注法定节假日与调休；右上角图例显示当月
各类事件数量；点某一天可查看和添加当天日程。

### 办案日志与邮寄记录
每个案件可记录办案日志（含工时）和文书邮寄记录（收件人、内容、快递单号），随手沉淀为台账，
可一键复制导出。

### 期限提醒
按案件开关。开启后自动扫描封皮里的日期，临期（默认 7 天内）在顶栏铃铛亮红点并弹系统通知。

### 律师工具箱
顶栏工具箱图标，离线计算：诉讼费、财产保全费、风险代理费、仲裁费（费率可校准）、法定期限
届满日（自动顺延节假日）、金额大写、文本去重、字数统计。

### 备份与安全
- 每 12 小时自动备份全部数据；结构升级前自动留底；保留最近多份。
- 「设置 → 案件管理 → 备份与恢复」可查看、恢复、导入、导出备份。
- 数据库存于系统用户资料目录（macOS：`~/Library/Application Support/MatterVibe/`；
  Windows：`%AppData%\MatterVibe\`），不在程序目录，卸载或升级软件不会丢数据。

### 多设备同步（Mac ↔ Windows）
「设置 → 设备同步」，两种方式：
- **坚果云自动同步**：配置一次（坚果云账户 + 应用密码 + 自设加密口令），打开软件即自动把本机
  改动加密上传、拉取其他设备改动合并。端到端加密（AES-256-GCM），云端只存密文。
- **手动同步包**：导出一个 .json 文件，拷到另一台设备导入合并。
- 两台设备必须使用**相同的加密口令**。口令遗忘则云端数据无法恢复（本地数据不受影响），
  请妥善保管。

### 本地 AI 接口（可选，默认关闭）
「设置 → AI 接口」开启后，支持的 AI 助手（如 Marvis 等）可通过本机接口读取案件材料、帮你
录入看板。接口仅监听本机 127.0.0.1、需令牌鉴权、可设为只读。涉密案件建议保持关闭或只读。
详见软件目录下《AI接口说明.md》。

## 四、数据安全与隐私

- 所有数据存在本机，不向任何服务器上传（除非你主动开启坚果云同步，且同步数据经端到端加密）。
- AI 接口默认关闭；开启后 AI 读到的内容会发送给其大模型服务商，故涉密案件请关闭或只读。
- 建议给电脑设开机密码：本机数据库未加密，电脑本身的安全即数据的安全边界。

## 五、常见问题

**Q：升级软件会丢数据吗？** 不会。数据存在用户资料目录，与程序分离；升级前还会自动备份。

**Q：换了新电脑怎么办？** 用「设备同步」（坚果云或手动同步包）把数据迁移过去；或直接复制
用户资料目录下的数据库文件。

**Q：模板能改吗？** 能。在「模板编辑」里自定义，也能导入别人分享的模板。

**Q：是免费的吗？** 是。开源、免费，按 Apache License 2.0 提供。

**Q：遇到问题或想提建议？** 见 README 中的联系方式与项目地址。
