# MatterVibe 安装说明（给使用者）

> 你拿到的 MatterVibe 是一款免费、开源、数据全部存在自己电脑上的律师案件管理软件。
> 因为它没有花钱购买苹果/微软的官方签名认证，第一次打开时系统会拦一下——这是正常的，
> 按下面对应你电脑的步骤操作即可，软件本身安全。

---

## 一、我的电脑是 Mac

### 第 1 步：把软件拖进「应用程序」
打开下载到的 MatterVibe（如果是 .dmg，先双击打开它）。
**不要直接在那个窗口里双击运行**，先把 **MatterVibe** 图标用鼠标**拖进「应用程序 / Applications」**文件夹。
如果是 .dmg，拖完后把那个 .dmg 推出（Eject）。

### 第 2 步：解除"来自互联网"的安全限制
按 `Command + 空格`，搜索并打开「终端」。把下面这行**完整复制**进去，按回车：

```
sudo xattr -r -d com.apple.quarantine /Applications/MatterVibe.app
```

- 回车后会要你输入**这台 Mac 的开机密码**。注意：输密码时屏幕上不会显示任何字符（这是正常的），
  闭着眼输完直接回车即可。
- 这行命令的意思是告诉系统"这个软件可信，把苹果默认加的隔离标签去掉"。

做完后，去启动台（Launchpad）双击打开 **MatterVibe** 就能正常用了。若还弹一次"从互联网下载，
是否确定打开"，点「打开」即可。

> 嫌命令麻烦？也可以：在「应用程序」里**右键点** MatterVibe →「打开」→ 弹窗里再点「打开」。
> 首次这样打开一次后，以后就能正常双击了。

### 注意芯片型号
- 如果你的 Mac 是 **Apple 芯片（M1/M2/M3/M4）**，下载文件名里带 **arm64** 的版本。
- 如果是较老的 **Intel 芯片** Mac，下载带 **x64 / intel** 的版本。
- （在 Mac 左上角苹果标 →「关于本机」可以看到芯片型号。）

---

## 二、我的电脑是 Windows

### 第 1 步：解压
把下载的压缩包解压到一个固定位置（比如 `D:\MatterVibe`）。**不要在压缩包内直接运行**，
先解压出来。

### 第 2 步：运行
进入解压后的文件夹，双击 **MatterVibe.exe**。

### 如果出现蓝色"Windows 已保护你的电脑"提示
这是因为软件没买微软的签名认证，正常现象。点提示框里的 **「更多信息」**，
再点出现的 **「仍要运行」** 即可。只有第一次需要这样，之后正常双击。

### 注意系统类型
- 绝大多数 Windows 电脑下载带 **x64** 的版本。
- 如果你的是 ARM 架构的 Windows（较少见），下载带 **arm** 的版本。

---

## 三、关于数据安全

- MatterVibe 的所有案件数据只存在你自己的电脑上，不会上传到任何服务器。
- 软件每隔一段时间自动备份；卸载或更新软件不会丢数据。
- 建议给电脑设置开机密码——电脑本身的安全，就是案件数据的安全。

---

## 四、第一次使用

打开软件后会有一个简短的引导页。点「＋ 新建案件」选一套模板就能开始；
需要帮助时，点右上角设置 →「使用帮助」随时查看。

有问题可联系提供软件给你的人，或访问项目主页。
