# MatterVibe · Mac 安装指南（发给他人使用）

> 这是一款用 AI 独立开发、未经苹果付费公证的自用软件，因此首次打开会被 macOS 的
> 安全机制拦一下。按下面两步即可正常使用，整个过程安全。

## 第一步：把软件拖进「应用程序」

打开下载到的 `.dmg` 后，**不要**直接在 DMG 窗口里双击图标运行。

- 先把 **MatterVibe** 图标用鼠标拖进「应用程序 / Applications」文件夹
- 拖完后关闭并推出（Eject）那个 .dmg 磁盘映像

## 第二步：解除「来自互联网」的隔离标记

打开「终端」（按 `Command + 空格` 搜索"终端"即可），复制下面这行粘贴进去、按回车：

```bash
sudo xattr -r -d com.apple.quarantine /Applications/MatterVibe.app
```

- 回车后会要求输入你的 **Mac 开机密码**（输入时屏幕不显示任何字符，是正常的，
  盲输完直接回车即可）
- 这行命令的作用：告诉系统"这个软件是可信的，撕掉苹果默认附加的隔离标签"

完成后，去启动台（Launchpad）双击打开 **MatterVibe**。若再弹出"从互联网下载的应用，
是否确定打开"，点「打开」即可正常使用。

---

如果嫌命令麻烦，也可以用更简单（但要点两次）的方式：在「应用程序」里**右键点击**
MatterVibe →「打开」→ 在弹窗里再点「打开」。首次这样打开后，以后就能正常双击了。
