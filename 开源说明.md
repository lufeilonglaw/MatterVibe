# MatterVibe 开源说明

MatterVibe 采用 **Apache License 2.0** 开源。完整法律文本见随附的 `LICENSE` 文件；本文用
通俗语言说明它意味着什么，以及本项目的开源姿态。

## 为什么选 Apache License 2.0

Apache 2.0 是一份宽松、对使用者友好、同时对作者有基本保护的开源许可证，被大量主流项目采用。
相比 GPL 等"传染性"许可（要求基于它的衍生作品也必须开源），Apache 2.0 不强制传染——
别人可以基于 MatterVibe 开发自己的（哪怕闭源的、商用的）软件。这符合本项目"温和开放、
鼓励创造"的姿态：希望同行能自由使用、改造、再分发，而不是用许可证去约束大家。

## 你可以做什么（任何人）

- **自由使用**：个人、律所、商业环境都可以用，不收费、无需授权。
- **自由修改**：可以改源代码，做成你自己的版本。
- **自由分发**：可以把原版或你的修改版分发给别人，包括用于商业目的。
- **自由二次开发**：可以基于它开发新软件，你的新软件可以选择任何许可证（包括闭源）。

## 你需要遵守什么

- **保留版权与许可声明**：分发时要保留原有的版权声明、LICENSE 文件和 NOTICE 文件。
- **声明改动**：如果你改了源文件，需要在被改动的文件中标注"已修改"。
- **不得使用作者商标背书**：不能用原作者的名义为你的衍生版本做担保或背书。
- **NOTICE 文件**：分发时需附带 NOTICE 文件（其中是作者署名信息）。
